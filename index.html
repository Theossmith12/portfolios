My Projects

UK Crime Data Dashboard

This platform, currently in progress as part of my dissertation, aims to enable police and law enforcers to anticipate crimes by type with high geographic precision. It analyzes UK police street-level crime data and presents the insights through an intuitive, user-friendly dashboard. Key features already developed include geographic crime heatmaps, crime type filtering, and temporal analysis tools.

Aqua Insight

A Python-based tool for analyzing air quality data from the LondonAir API. This project computes key metrics like daily and hourly pollutant averages, peak pollution hours, and trends over time. It includes data cleaning, real-time visualization, and ASCII-based graphing, offering insights for environmental monitoring and public health.

Learning from Data: Credit Card Fraud Detection

A comprehensive exploration of credit card fraud detection using machine learning techniques and imbalanced data handling methods. This project showcases key results, evaluation metrics, and an interactive Jupyter Notebook for deeper insights.
