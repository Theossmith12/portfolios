Aqua Insight: Real-Time Air Quality Monitoring

Aqua Insight is a Python-based tool for analyzing air quality data from London's monitoring stations. This project leverages the LondonAir API to calculate key metrics, including daily and hourly averages, peak pollution hours, and visual trends over time. It provides valuable insights into environmental health, helping inform decisions on air quality management.

Key Features

  • Real-time data analysis from the LondonAir API.
  • Daily, hourly, and monthly pollution averages.
  • Graphical visualization of air quality trends.
  • Handling and interpolation of missing data points.
  • Peak pollution hour identification.
